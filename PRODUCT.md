# Product

## Register

product

## Users

面向需要直连 MySQL 实例的开发者、测试人员和系统管理员。他们在本地开发或受控的内网环境中，需要快速查看、创建和维护数据库与数据表，同时避免命令行操作中的误删风险。

## Product Purpose

Mysqladmin 是一个 Spring Boot 3 与 Vue 3 构建的网页端 MySQL 管理工具。它让用户建立动态连接、浏览数据库和表结构、管理库表基础 CRUD，并在高风险操作前明确确认。

## Brand Personality

清醒、专业、敏捷。界面应有现代开发工具的效率与精致感，但不能套用老式企业后台或泛化 AI 模板。

## Anti-references

避免 phpMyAdmin 的传统密集视觉、灰暗陈旧的企业系统、夸张渐变和大面积玻璃拟态，以及没有信息层级的卡片拼贴。

## Design Principles

- 让数据库状态和当前操作始终一目了然。
- 将专业能力隐藏在清晰的渐进式操作中。
- 高风险操作必须醒目、可撤回地确认。
- 一致的密度和组件语义优先于装饰。
- 让熟悉数据库工具的用户不需要学习界面。

## Accessibility & Inclusion

以 WCAG 2.1 AA 为基线：文本保持足够对比度，键盘可访问，焦点状态清晰，并尊重减少动态效果偏好。
