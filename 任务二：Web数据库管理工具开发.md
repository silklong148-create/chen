# 任务二：Web数据库管理工具开发

## 一、任务目标

1\. 基于 SpringBoot3 \+ Vue3 前后端分离架构，开发一款网页版 MySQL 数据库可视化管理工具。

2\. 实现网页端动态连接 MySQL 服务器，支持数据库、数据表的完整增删改查管理功能。

3\. 掌握动态 JDBC 连接、动态 SQL 执行、数据库元数据读取、前后端动态交互开发能力。

4\. 强化数据库底层原理、Web 动态操作数据库、接口安全校验的综合实战能力。

## 二、任务准备

1\. 已完成任务一环境：JDK17/21、Maven/Gradle、nvm、Vue3 开发环境、IDEA/VS Code 及 AI 辅助插件。

2\. 本地或远程 MySQL 服务，具备正常连接权限，可用于测试数据库、数据表操作。

3\. 掌握 MySQL 基础 DDL、DML 语句，熟悉数据库、数据表结构特性。

## 三、具体任务内容

**参考案例：php\-admin**，仅参考该工具的核心功能、交互逻辑与业务架构进行开发实现，**不参考其页面视觉样式**，前端页面需自主设计优化，保证界面美观高级。

**开发辅助工具要求**：项目开发全程推荐安装并使用 **mysql\_mcp\_server**，依托 MCP 协议实现 AI 工具与 MySQL 数据库联动，辅助完成 SQL 编写、数据库调试、表结构设计、问题排查，提升开发效率与代码规范性。

### 1\. MySQL 服务器连接功能

1\. 页面提供数据库连接表单，支持填写地址、端口、账号、密码。

2\. 后端实现动态 JDBC 连接校验，支持连接测试，返回连接成功或失败信息。

3\. 会话保持连接状态，切换功能无需重复连接，保证操作连贯性。

### 2\. 数据库层级管理（增、删、改、查）

连接 MySQL 服务后，支持对实例下所有数据库进行管理：

1\. 数据库查询：读取当前 MySQL 实例所有数据库列表，展示基础信息。

2\. 数据库新增：支持自定义数据库名称、字符集、排序规则，在线创建数据库。

3\. 数据库修改：支持修改数据库字符集、排序规则等配置信息。

4\. 数据库删除：支持删除指定数据库，高危操作增加二次确认弹窗。

### 3\. 数据表层级管理（增、删、改、查）

选中指定数据库后，对库内数据表进行可视化全功能管理：

1\. 数据表查询：展示当前库下所有数据表列表，支持查看表结构、字段详情、表数据。

2\. 数据表新增：可视化表单创建数据表，可配置字段名、字段类型、长度、主键、自增、非空、默认值、注释等。

3\. 数据表修改：支持新增字段、删除字段、修改字段属性、修改表名与表注释。

4\. 数据表删除：支持删除指定数据表，高危操作增加二次确认弹窗。

### 4\. 进阶数据库功能开发

在基础库表增删改查功能之上，拓展高级数据库管理能力，适配主流数据库进阶操作：

1\. **视图管理功能**：支持自定义创建数据库视图、查看已有视图结构与数据、编辑视图语句、删除视图，实现视图全生命周期可视化管理。

2\. **索引管理功能**：支持为数据表创建普通索引、唯一索引、主键索引，可查看表所有索引详情、编辑索引属性、删除冗余索引，优化数据库查询性能。

3\. **多数据库适配（SQL Server）**：工具不仅支持 MySQL，拓展适配 SQL Server 数据库连接与管理，可动态连接 SQL Server 服务，实现库、表、视图、索引的基础管理操作，完成多数据库兼容适配。

## 四、任务要求

1\. 前后端代码分层规范，数据库操作统一封装，代码复用性高、结构清晰。

2\. 做好 SQL 安全过滤，防止 SQL 注入，保障动态执行 SQL 的安全性。

3\. 所有删除、修改等高风险操作必须添加二次确认机制，避免误操作。

4\. 页面交互友好，操作反馈完善，所有操作均有成功/失败提示。

5\. 项目可完整本地启动运行，数据库、数据表所有增删改查功能正常可用。

6\. 前端页面设计整洁美观、布局规整，**拒绝通用 AI 模板风格**，参考 php\-admin 商业化写实管理界面视觉，配色协调、组件统一、交互舒适，具备正式项目页面质感。
