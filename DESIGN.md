# Design System: Mysqladmin

## Overview

一个明亮、克制的开发者工具界面。它像一张整洁的工作台：纯白内容区承载表格与结构信息，低饱和青蓝色只用于当前选择和主要操作，深色墨蓝提供稳定的代码与数据语境。

## Color

所有颜色使用 OKLCH。

```css
:root {
  --bg: oklch(1 0 0);
  --surface: oklch(0.975 0.006 215);
  --surface-strong: oklch(0.944 0.012 215);
  --ink: oklch(0.23 0.025 245);
  --muted: oklch(0.49 0.028 240);
  --primary: oklch(0.52 0.13 200);
  --primary-hover: oklch(0.45 0.13 200);
  --accent: oklch(0.63 0.16 150);
  --danger: oklch(0.56 0.19 25);
  --warning: oklch(0.72 0.16 78);
  --border: oklch(0.88 0.012 220);
}
```

## Typography

使用 `Inter, "Segoe UI", system-ui, sans-serif`。产品界面使用固定字号层级：12、13、14、16、20、24px；表格及元信息使用 13px。

## Layout

- 左侧 252px 浅灰数据库资源导航，参考数据库树的紧凑浏览方式；连接状态置顶、数据库列表保持清晰的当前项状态，小屏幕折叠为横向导航。
- 顶部 76px 工具栏显示当前位置、当前服务端标识与主要操作；对象信息以一行技术元数据呈现。
- 主内容区以标题、对象页签、工具栏、斑马纹数据表的节奏呈现，保证高信息密度同时避免无意义的嵌套卡片；选中态与数据表行悬停态只使用主色。
- 删除和破坏性操作使用原生语义对话框与清晰的危险色。

## Components

- 12px 圆角用于面板，8px 圆角用于控件。
- 主按钮为青蓝实色配白字；次操作保持轻量边框。
- 表格行有清晰悬停和选中态；焦点使用 2px 主色外环。
- 动效仅用于面板切换、提示与加载状态，时长 160–220ms，并提供 reduced-motion 降级。
