# 任务一：AI代码助手 \+ SpringBoot3 \+ Vue3 开发环境搭建与基础入门

## 一、任务目标

1\. 掌握 Claude Code / Codex 工具安装、大模型接口配置，熟练使用基础AI编程命令。

2\. 完成 Java 高版本环境、构建工具、前端 Node 多版本环境搭建，实现前后端基础开发环境就绪。

3\. 熟悉 SpringBoot3 \+ Vue3 技术栈基础开发环境组成，掌握必备工具与插件配置。

4\. 安装配置主流AI编程辅助插件，建立高效的AI辅助开发工作流。

## 二、任务准备

1\. 正常联网 Windows / Mac / Linux 电脑，熟悉基础终端、命令行操作。

2\. 准备可用大模型账号或 API Key，用于 AI 代码工具联网调用。

3\. 安装 IDEA 或 VS Code 主流开发编辑器。

## 三、具体任务内容

### 任务1：AI代码助手部署与使用（Claude Code / Codex 二选一）

根据个人电脑环境任选其一安装配置，完成基础命令学习。

1\. 本地安装 Claude Code 或 Codex 工具，终端校验安装成功、无报错。

2\. 配置大模型环境：支持账号直连或 API 接口配置两种方式任选其一，确保模型调用正常。

3\. 熟悉工具基础命令：掌握版本查看、环境检测、代码生成、代码解释、纠错优化、帮助查询等常用命令，可独立在终端执行使用。

### 任务2：Java 开发环境配置（JDK 17 / 21）

完成 SpringBoot3 适配的高版本 JDK 环境安装与全局配置，推荐 JDK17 或 JDK21。

1\. 下载并安装 JDK17 或 JDK21 官方版本或开源适配版本。

2\. 配置系统环境变量 JAVA\_HOME、Path，保证全局终端可识别 java 命令。

3\. 终端执行 `java -version`、`javac -version` 校验环境生效，确保版本匹配、无环境冲突。

### 任务3：Java 项目构建工具基础（Maven / Gradle）

任选其一完成安装配置，掌握基础使用，了解 SpringBoot3 主流构建方式。

1\. 安装 Maven 或 Gradle，配置本地仓库、镜像源（国内镜像加速）。

2\. 配置系统环境变量，终端校验 `mvn -v` 或 `gradle -v` 正常输出版本信息。

3\. 掌握基础命令：项目构建、依赖下载、编译、运行、清理打包等常用指令。

### 任务4：前端多版本 Node 环境管理（nvm）

为 Vue3 开发搭建可灵活切换的 Node 环境，统一前端开发基础环境。

1\. 根据系统安装对应版本 nvm 工具，完成环境初始化。

2\. 通过 nvm 安装稳定版 Node\.js 与 npm，设置默认使用版本。

3\. 掌握 nvm 常用命令：版本安装、版本切换、查看已安装版本、卸载版本等，保障 Vue3 项目正常运行。

### 任务5：开发工具 AI 插件安装与配置

基于 IDEA / VS Code 安装主流 AI 编程辅助插件，搭建智能开发环境。

1\. 熟悉插件基础功能：代码补全、代码生成、代码解释、错误修复、注释生成、重构优化。

2\. 简单测试插件生效，确保插件无冲突、可正常调用AI辅助能力。

### 任务6：SpringBoot3 \+ Vue3 开发环境就绪

整合以上所有环境，保证前后端基础开发环境完整可用。

1\. 确认 JDK、构建工具、前端 nvm/node 环境全部配置生效。

2\. 熟悉 SpringBoot3 项目基础运行环境要求与 Vue3 项目启动依赖环境。

3\. 理解前后端分离开发的基础环境栈组成，为后续项目开发、联调、部署做好铺垫。

## 四、任务要求

1\. 独立完成全部环境搭建与配置操作，熟悉每一项环境的作用与配置逻辑。

2\. 独立排查解决安装、配置、命令使用过程中的报错问题，培养自主问题排查能力。

3\. 最终确保 AI 代码工具、Java 环境、前端环境、编辑器插件全部正常可用、无基础报错。
